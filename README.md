# Qurbani Online Bazar

##  Project Purpose

Qurbani Online Bazar is a modern online marketplace where users can browse, search, and explore different Qurbani animals for Eid-ul-Adha. Users can view animal details, manage profiles, and experience a clean and responsive interface.

---

##  Live URL

https://your-live-link.com

---

##  Key Features

*  Authentication system using Better Auth
*  User profile and profile update feature
*  Browse all Qurbani animals
*  Animal details page with dynamic routing
*  Sorting animals by price
*  Customer testimonials section
*  Fully responsive design
*  Modern UI with Tailwind CSS and DaisyUI
*  Smooth animations using Animate.css
*  Toast notifications using React Toastify

---

##  Technologies Used

* Next.js
* React
* Tailwind CSS
* DaisyUI
* Better Auth
* MongoDB
* React Hook Form
* React Icons
* React Toastify
* Animate.css

##  NPM Packages Used

```bash
npm install next react react-dom
npm install tailwindcss @tailwindcss/postcss postcss
npm install daisyui
npm install better-auth
npm install mongodb
npm install react-hook-form
npm install react-icons
npm install react-toastify
npm install animate.css
```
